# GAME SPEC — ระบบอาชีพ + ข่าว + ลูปการเล่น
โมดูลบลัฟอาชีพสำหรับเกมลงทุน multiplayer (สเปกนี้ครอบคลุมเฉพาะส่วนอาชีพ/ข่าว/ลูปเทิร์น — ระบบตลาด ราคา ธนาคาร ธุรกิจ ใช้ของเกมหลักที่มีอยู่แล้ว)

> **ไม่มี "กองกลาง" (central pot)** — เงินทุกรายการไหลระหว่างผู้เล่นโดยตรง
> หรือเข้า/ออกจาก "ระบบ" (money sink/source)

---

## 1. ภาพรวม

- ผู้เล่น 4–6 คน เล่นเป็นรอบ วนทีละคน
- ทุกคนเริ่มด้วยเงินสด **$10,000** เท่ากัน
- แต่ละคนถือ **การ์ดอาชีพลับ 2 ใบ** (สุ่มจากกอง 24 ใบ = 8 อาชีพ × อาชีพละ 3 ใบ — ซ้ำกันได้ทั้งในมือตัวเองและข้ามผู้เล่น)
- **ไม่มีการสรุปผล/จัดอันดับระหว่างเกม** — ตัดสินครั้งเดียวตอนจบเกม
- **ผู้ชนะ = รวมมูลค่าสินทรัพย์สูงสุดตอนจบเกม**
  (เงินสด + เงินฝาก + มูลค่าสินทรัพย์ทุกตัว ณ ราคาปิด + มูลค่าธุรกิจ − หนี้คงค้าง)
- จอสรุปท้ายเกม: อันดับมูลค่า + เฉลยอาชีพทุกคน + ไทม์ไลน์การโกหก/challenge ทั้งเกม

---

## 2. ระบบอาชีพ

### 2.1 การแจกและถือ
- กอง 24 ใบ สุ่มแจกคนละ 2 ใบ **ลับเฉพาะเจ้าตัว** ที่เหลือคว่ำเป็นกองจั่ว
- ซ้ำได้ทุกแบบ: มือเดียวถืออาชีพเดียวกัน 2 ใบได้ / ต่างคนถืออาชีพเดียวกันได้

### 2.2 การใช้ท่า
- ใช้ท่าโจมตีได้ **1 ครั้งต่อเทิร์นตัวเอง** หรือเลือก "ข้าม" รับเงินเดือน $300 จากระบบ
- ตอนใช้ท่า **ประกาศอาชีพ** — จะเป็นอาชีพที่ถือจริงหรืออ้างมั่วก็ได้
- เคาน์เตอร์ใช้ได้ **เฉพาะเมื่อตัวเองเป็นเป้าถูกโจมตี** และเฉพาะกับ **ท่าของอาชีพนักล่าตามวงจร** เท่านั้น — เคาน์เตอร์ก็อ้างมั่วได้

### 2.3 วงจรนักล่า (ใครเคาน์เตอร์ท่าใครได้)

```
สื่อ → วาฬ → ชอร์ต → เสือ → แบงค์ → มาเฟีย → ก.ล.ต. → นักสืบ → กลับไปสื่อ
```

ลูกศรชี้จาก "นักล่า" ไปหา "ผู้ที่เคาน์เตอร์กลับได้"
เช่น เมื่อ **สื่อ** ใช้ท่าใส่ใคร เป้า **อ้างวาฬ** เพื่อเคาน์เตอร์ได้

**ผลเคาน์เตอร์ (มาตรฐานเดียวทุกอาชีพ):** ท่าโจมตียกเลิก + ผู้โจมตีจ่ายเป้า **$500**

### 2.4 Challenge (จับโกหก)
- ถูกจับได้ว่า **อ้างมั่ว** → ท่ายกเลิก + จ่ายคนที่ challenge **$800** + เปิดการ์ดทิ้งถาวร 1 ใบ (เลือกเอง) + ติดสถานะเครดิตพัง 2 รอบ
- **จับผิด** (เขาอ้างจริง) → คนจับจ่ายเจ้าตัว **$800** / เจ้าตัวเปิดใบพิสูจน์แล้ว **สับกลับกอง จั่วใบใหม่** (ตัวตนกลับมาลับ) / ท่าทำงานต่อตามปกติ
- ผู้เล่นที่เปิดทิ้งครบ 2 ใบ: ใช้ท่าโจมตี/เคาน์เตอร์ไม่ได้ถาวร แต่ยัง challenge, เดินแมพ, ลงทุนได้ปกติ — ยังชนะเกมได้ด้วยพอร์ต
- การ์ดที่ "เปิดเผยหน้า" จากท่าข่มขู่ของมาเฟีย ไม่นับเป็นการทิ้ง — ยังใช้งานได้ แค่ไม่ลับแล้ว

## 3. รายละเอียด 8 อาชีพ

> ทุกท่าโจมตีเลือกเป้าได้ **1 คนเท่านั้น** ไม่มีท่าใดกระทบบุคคลที่สามหรือราคาตลาดกลาง

### 3.1 เจ้าพ่อสื่อ (Media Tycoon)
- **⚔️ ยัดข่าวปลอม** — เป้า: ผู้เล่นที่มีการ์ดข่าววงใน
  - ระบบสุ่มการ์ดข่าวของเป้า 1 ใบให้ผู้โจมตีดู → ผู้โจมตีเลือก: **สลับ**ใบนั้นกับใบบนสุดของกองข่าว หรือ **วางคืน**
  - เป้าได้แจ้งเตือนแค่ว่า "คุณถูกยัดข่าวปลอม" — ไม่รู้ว่าใบไหนถูกดู ไม่รู้ว่าถูกสลับหรือไม่
  - ถ้าเป้าไม่มีการ์ดข่าว: ผู้โจมตีรู้ว่าเป้าไม่มีข่าว และจั่วข่าวเอง 1 ใบฟรี
- **🛡️ เคาน์เตอร์ได้เมื่อโดน:** นักสืบ "เจาะระบบ"

### 3.2 วาฬคริปโต (Crypto Whale)
- **⚔️ เทใส่ (Dump On)** — เป้า: ผู้เล่นที่ถือคริปโต (ระบบ disable เป้าที่ไม่มีคริปโต)
  - มูลค่าคริปโตในพอร์ตของเป้าลด **20%** (mark-down เฉพาะพอร์ตเป้า ไม่กระทบราคาตลาดกลาง)
  - ครึ่งหนึ่งของที่หาย (=10% ของมูลค่าเดิม) โอนเป็นเงินสดเข้ากระเป๋าวาฬ อีกครึ่งหายเข้าระบบ
- **🛡️ เคาน์เตอร์ได้เมื่อโดน:** สื่อ "ยัดข่าวปลอม"

### 3.3 สายชอร์ต (Short Seller)
- **⚔️ ชอร์ตพอร์ต** — เป้า: ผู้เล่นที่มีสินทรัพย์หรือเงินฝาก (มีแต่เงินสดล้วน = disable)
  - เป้าจ่ายผู้โจมตี **10% ของมูลค่าสินทรัพย์ประเภทที่เป้าถือมูลค่าสูงสุด** (เพดาน $1,500) — นับเงินฝากเป็นสินทรัพย์ด้วย
  - เงินสดเป้าไม่พอ → ระบบบังคับขายสินทรัพย์นั้นที่ราคาตลาด −10% จนพอ
- **🛡️ เคาน์เตอร์ได้เมื่อโดน:** วาฬ "เทใส่"

### 3.4 เสือนอนกิน (Property Mogul)
- **⚔️ ฮุบที่ดิน** — เป้า: ผู้เล่นที่ถืออสังหา (ผู้โจมตีต้องมีเงินสดพอจ่าย ไม่งั้น disable)
  - บังคับซื้ออสังหาของเป้า 1 หน่วยที่ **ราคาตลาด −15%** (เงินจากผู้โจมตี → เป้า, อสังหาย้ายเข้าพอร์ตผู้โจมตี)
  - เป้าปฏิเสธได้โดยจ่ายค่ายอมความ **$500 ให้ผู้โจมตี** แทน
- **🛡️ เคาน์เตอร์ได้เมื่อโดน:** ชอร์ต "ชอร์ตพอร์ต"

### 3.5 นายแบงค์ (Banker)
- **⚔️ บีบหนี้** — เป้า: ผู้เล่นที่มีเงินกู้ค้าง (ไม่มีหนี้ = disable)
  - เป้าชำระหนี้คืนระบบทันที **50% ของยอดคงค้าง**
  - เงินสดไม่พอ → เป้าเลือกขายสินทรัพย์เองที่ราคาตลาด −10% จนครบ
- **🛡️ เคาน์เตอร์ได้เมื่อโดน:** เสือ "ฮุบที่ดิน"

### 3.6 มาเฟียเงินกู้ (Loan Shark)
- **⚔️ ข่มขู่** — เป้า: ผู้เล่นคนใดก็ได้
  - เป้าเลือก 1: **จ่ายผู้โจมตี $1,000** หรือ **เปิดการ์ดอาชีพ 1 ใบ** (เป้าเลือกใบเอง เปิดให้ทุกคนเห็นถาวรแต่ไม่ทิ้ง)
  - เงินสดเป้าไม่ถึง $1,000 → บังคับเปิดการ์ดอย่างเดียว
- **🛡️ เคาน์เตอร์ได้เมื่อโดน:** แบงค์ "บีบหนี้"

### 3.7 ก.ล.ต. (Regulator)
- **⚔️ อายัดบัญชี** — เป้า: ผู้เล่นคนใดก็ได้
  - เทิร์นถัดไปของเป้า: ห้ามซื้อขาย / กู้ / ฝากถอน / เข้าตึกข่าว / ใช้ท่าโจมตี — เดินแมพได้อย่างเดียว
  - เป้ายังคง challenge และเคาน์เตอร์ได้ถ้าถูกโจมตีระหว่างติดอายัด
- **🛡️ เคาน์เตอร์ได้เมื่อโดน:** มาเฟีย "ข่มขู่"

### 3.8 นักสืบไซเบอร์ (Cyber Investigator)
- **⚔️ เจาะระบบ** — เป้า: ผู้เล่นคนใดก็ได้ / เลือกโหมด:
  - **โหมดเงิน:** ขโมยเงินสด $800 จากเป้า (มีไม่ถึงเอาเท่าที่มี)
  - **โหมดข้อมูล:** ระบบสุ่มเปิดการ์ดอาชีพเป้า 1 ใน 2 ใบให้ผู้โจมตีดูคนเดียว + ส่งสรุปพอร์ตเต็มของเป้า (สินทรัพย์/เงินสด/เงินฝาก/หนี้/ธุรกิจ) ให้ผู้โจมตีคนเดียว
- **🛡️ เคาน์เตอร์ได้เมื่อโดน:** ก.ล.ต. "อายัดบัญชี"

## 4. ระบบข่าว

### 4.1 ข่าวสาธารณะ
- เปิดใหม่ทุกต้นรอบ 1–2 ใบ ทุกคนเห็น ระบุ: สินทรัพย์เป้าหมาย + ทิศทาง + ขนาด (%) + รอบที่มีผล
- ข่าวสาธารณะจากระบบมีผลต่อราคาจริงเสมอ (การขยับราคาใช้กลไกตลาดของเกมหลัก)
- **ข่าวลือที่ผู้เล่นจ่ายปล่อย** (ผ่านตึกข่าว) แสดงปนกับข่าวสาธารณะโดยแยกไม่ออก — แต่**ไม่มีผลต่อราคา**เลย

### 4.2 ข่าววงใน (การ์ดลับ)
- แต่ละใบระบุ: สินทรัพย์ / ทิศทาง+ขนาด / รอบที่มีผล / flag ลับ `is_real`
- อัตราส่วนในกอง: **จริง 60% : ปลอม 40%**
- แจกฟรีคนละ 1 ใบตอนเริ่มเกม / ที่เหลือได้จากตึกข่าวหรือท่าอาชีพ
- ใบจริงขยับราคาจริงเมื่อครบกำหนด ใบปลอมเฉลยแล้วไม่มีอะไรเกิดขึ้น (ระบบประกาศเฉลยจริง/ปลอมให้ทุกคนเห็น)

### 4.3 ตึกข่าวสาร (สถานที่บนแมพ — ใช้บริการได้ 1 อย่าง/การเข้า 1 ครั้ง)

| บริการ | ราคา (จ่ายเข้าระบบ) | ผล |
|---|---|---|
| ซื้อข่าววงใน | $500 | จั่วการ์ดข่าววงใน 1 ใบ |
| ตรวจสอบข่าว (fact-check) | $700 | เลือกข่าวในมือ 1 ใบ ระบบบอกเจ้าตัวคนเดียวว่า จริง/ปลอม |
| ปล่อยข่าวลือ | $1,000 | เลือกพาดหัวจาก template (สินทรัพย์+ทิศทาง+ขนาด) → แสดงเป็นข่าวสาธารณะรอบถัดไปแบบไม่ระบุตัวตน ไม่มีผลต่อราคา |

- ระบบโชว์ให้ทุกคนเห็นว่า "ใครเดินเข้าตึกข่าว" แต่**ไม่โชว์ว่าใช้บริการไหน**

## 5. ลูปการเล่น

### 5.1 โครงรอบ
เล่นทีละคนตามลำดับ (สุ่มตอนเริ่มเกม แล้วคงที่) → ครบทุกคน = จบรอบ → ระบบประมวลผลท้ายรอบ**อัตโนมัติเบื้องหลังทันที ไม่มีหน้าจอรอ/หน้าสรุป** (ราคาขยับ ข่าวครบกำหนดมีผล+เฉลย ดอกเบี้ย/ค่าเช่า/รายได้ธุรกิจ ตัวนับสถานะลด) → เข้ารอบใหม่ต่อเนื่อง — ผู้เล่นเห็นความเปลี่ยนแปลงเองจากกราฟราคาและ HUD / สรุปผลมีครั้งเดียวคือจอจบเกม

### 5.2 เทิร์นของผู้เล่น 1 คน

```
① เฟสโจมตี — ไม่จับเวลา
   ผู้เล่นคิดได้เต็มที่ เลือกอย่างใดอย่างหนึ่ง:
   a) ใช้ท่าโจมตี: ประกาศอาชีพ + เลือกเป้า → เข้าสู่ resolution (5.3)
   b) ข้าม: รับเงินเดือน $300 จากระบบ
   * การ resolve ทั้งหมดที่ตามมา (counter / challenge ที่ตัวเองโดน)
     เกิดในช่วงไม่จับเวลานี้ — ไม่กินเวลาเฟสแมพ

② เฟสแมพ — จับเวลา 60 วินาที (เริ่มนับหลัง resolution เฟสโจมตีจบสนิท)
   ภายใน 60 วิ ทำอะไรก็ได้ กี่อย่างก็ได้ ไม่มีระบบแต้ม —
   ข้อจำกัดเดียวคือ "เวลาจริง": การเดินระหว่างสถานที่ + การกดทำธุรกรรมในตึก
   ล้วนกินเวลาใน 60 วินี้ทั้งหมด
   - หมดเวลา = จบเทิร์นทันที ไม่ว่ากำลังทำอะไรค้างอยู่ (ธุรกรรมที่ยังไม่กดยืนยัน = ไม่เกิด)
   - avatar เดินบนแมพด้วยความเร็วคงที่ (ระยะทางจึงเป็นต้นทุนจริง —
     วางแผนเส้นทางไม่ดีก็เสียเวลาเปล่า)
   - ผู้เล่นอื่นเห็น avatar เดินเข้าที่ไหน แต่ไม่เห็น transaction ใดๆ
   - เข้าตึกแล้ว "ไม่ทำอะไร" แล้วเดินออกก็ได้ (เผาเวลาตัวเองเพื่อหลอกคนอื่น)

   สถานที่: ตลาดหุ้น / ศูนย์คริปโต / ร้านทอง / สำนักงานที่ดิน /
            ธนาคาร / ออฟฟิศธุรกิจ / ตึกข่าวสาร / ลีดเดอร์บอร์ด

   กิจกรรมต่อสถานที่ (ระหว่างอยู่ในตึก ทำกี่ธุรกรรมก็ได้ตราบที่เวลายังเหลือ):
   - ตลาดหุ้น/ศูนย์คริปโต/ร้านทอง/สำนักงานที่ดิน: ซื้อ/ขายกี่หน่วยก็ได้
   - ธนาคาร: กู้ / ชำระหนี้ / ฝาก / ถอน
   - ออฟฟิศธุรกิจ: ตั้งหรือขยายธุรกิจ
   - ตึกข่าวสาร: ใช้บริการได้ (จ่ายเงินตามตาราง 4.3)
   - ลีดเดอร์บอร์ด: ส่องพอร์ตละเอียดของผู้เล่นทีละ 1 คน
     (เห็น: สินทรัพย์รายตัว / เงินสด / เงินฝาก / หนี้ / ธุรกิจ — ไม่เห็นการ์ดอาชีพและการ์ดข่าว)
     ต้นทุนคือเวลาที่เสียไปกับการเดินมา+เปิดดู แทนที่จะเอาไปลงทุน

③ จบเทิร์น → ผู้เล่นถัดไป
```

- **ข้อมูลสาธารณะฟรีตลอดเวลา:** อันดับมูลค่าสุทธิแบบหยาบ (ลำดับ 1–6 ไม่โชว์ตัวเลข) — ดูละเอียดต้องเดินไปที่ลีดเดอร์บอร์ดในเทิร์นตัวเอง (แลกด้วยเวลาใน 60 วิ)

### 5.3 Resolution ตอนมีการโจมตี

```
ผู้โจมตี: ประกาศอาชีพ X + เป้า   (เทิร์นผู้โจมตี — ไม่จับเวลา)
   │
   ▼
เป้าได้ popup ตัดสินใจภายใน 10 วินาที:
   [ยอมโดน]     → ท่ามีผลทันที
   [Challenge!] → ไปขั้น A
   [Counter]    → ไปขั้น B (ปุ่ม active เฉพาะเมื่อท่าที่โดนเป็นท่าของนักล่าตามวงจร)
   (หมดเวลา 10 วิ = ยอมโดนอัตโนมัติ)

A) Challenge — เป้ากล่าวหาว่าผู้โจมตีอ้างอาชีพมั่ว
   (ขั้นตอนเปิดการ์ด/จ่ายเงิน resolve ทันที ไม่มี timer — อยู่ในช่วงไม่จับเวลาของเทิร์นผู้โจมตี)
   - ผู้โจมตีถือการ์ด X จริง:
       → เปิดพิสูจน์ → คน challenge จ่ายผู้โจมตี $800
       → ผู้โจมตีสับใบที่เปิดกลับกอง จั่วใบใหม่
       → ท่าโจมตีทำงานตามปกติ
   - ผู้โจมตีอ้างมั่ว:
       → ท่ายกเลิก
       → ผู้โจมตีจ่ายคน challenge $800  ← เงินไปที่คนจับโกหกโดยตรง
       → ผู้โจมตีเปิดการ์ดทิ้งถาวร 1 ใบ (เลือกเอง)
       → ผู้โจมตีติดเครดิตพัง 2 รอบ

B) Counter — เป้าประกาศอาชีพเคาน์เตอร์ (ตามวงจรเท่านั้น)
   → ผู้โจมตีตัดสินใจ (เทิร์นตัวเอง — ไม่จับเวลา):
     [ยอม]        → ท่ายกเลิก + ผู้โจมตีจ่ายเป้า $500
     [Challenge!] → ตรวจการ์ดเป้า:
        - เป้าถืออาชีพเคาน์เตอร์จริง:
            → ท่ายกเลิก + ผู้โจมตีจ่ายเป้า $500 (ผลเคาน์เตอร์) + อีก $800 (แพ้ challenge)
            → เป้าสับใบพิสูจน์กลับกอง จั่วใหม่
        - เป้าอ้างมั่ว:
            → เคาน์เตอร์ล้ม ท่าโจมตีทำงานตามปกติ
            → เป้าจ่ายผู้โจมตี $800 + เปิดทิ้ง 1 ใบ + เครดิตพัง 2 รอบ

จบแค่ 2 ชั้นนี้ ไม่มี challenge ซ้อนเกินนี้ และบุคคลที่สามแทรกไม่ได้ทุกกรณี
```

## 7. โหมดเล่นกับบอท (Single Player — ทำก่อน Multiplayer)

> เป้าหมาย: ให้เกมเล่นจบได้ 1 คน + บอท 3–5 ตัว โดยใช้กติกาเดียวกับผู้เล่นจริงทุกประการ
> บอทเป็น rule-based ล้วน ไม่ต้องใช้ AI/ML — เน้นให้ "รู้สึกเหมือนคนที่มีนิสัย" ไม่ใช่เล่นเพอร์เฟกต์

### 7.1 กฎเหล็ก: บอทห้ามโกง
บอทเข้าถึงข้อมูลได้เท่าผู้เล่นจริงเป๊ะๆ:
- เห็น: ราคาตลาด, ข่าวสาธารณะ, feed เหตุการณ์ (ใครโจมตีใคร/ผล challenge), ตำแหน่ง avatar ทุกคน, อันดับหยาบ, การ์ดอาชีพ+ข่าวของ *ตัวเอง*
- ไม่เห็น: การ์ดอาชีพ/ข่าวของคนอื่น, พอร์ตละเอียดคนอื่น (ยกเว้นเดินไปส่องลีดเดอร์บอร์ดหรือใช้ท่านักสืบเหมือนผู้เล่น), flag จริง/ปลอมของข่าวตัวเอง (ยกเว้นจ่าย fact-check)
- ระบบสุ่ม/เฉลยทั้งหมดผ่าน game engine เดียวกับผู้เล่นจริง

### 7.2 บุคลิกบอท (สุ่มแจกตอนเริ่มเกม ไม่บอกผู้เล่น)

| บุคลิก | นิสัยลงทุน | นิสัยโจมตี | อัตราโกหก (อ้างอาชีพที่ไม่ถือ) | นิสัย challenge |
|---|---|---|---|---|
| **ฉลามเจ้าเล่ห์** | เก็งกำไรตามข่าว ถือคริปโต/หุ้นหนัก | โจมตีเกือบทุกเทิร์น เล็งคนอันดับสูงสุด | สูง (0.45) | challenge เมื่อมั่นใจ ≥ 0.7 |
| **เต่าปลอดภัย** | ทอง+อสังหา+เงินฝาก กู้น้อย | โจมตีน้อย ชอบข้ามรับเงินเดือน | ต่ำ (0.10) | แทบไม่ challenge (กลัวจ่าย $800) |
| **นักพนัน** | All-in ตามข่าววงในโดยไม่ fact-check ชอบกู้เต็มเพดาน | โจมตีสุ่มเป้า | กลาง (0.30) | challenge บ่อยเกินเหตุ (มั่นใจ ≥ 0.45) |
| **นักสังเกตการณ์** | ซื้อถัวเฉลี่ย สมดุลพอร์ต แวะลีดเดอร์บอร์ด+ตึกข่าวบ่อย | โจมตีเฉพาะเมื่อมีข้อมูล (เคยส่องพอร์ตเป้าแล้ว) | ต่ำ (0.15) | challenge แม่น — ใช้ระบบความจำ (7.3) เต็มที่ |

- เกม 1 ครั้งสุ่มบุคลิกไม่ซ้ำกันก่อน ถ้าบอทเกิน 4 ตัวค่อยซ้ำได้
- ทุกค่าความน่าจะเป็นใส่ noise ±0.05 ต่อเกม กันผู้เล่นจับแพทเทิร์นตายตัว

### 7.3 ระบบความจำของบอท (หัวใจของการ challenge ที่ดูฉลาด)
บอทเก็บบันทึกต่อผู้เล่นแต่ละคน (จากข้อมูลสาธารณะเท่านั้น):
- `claims[]` — เคยอ้างอาชีพอะไรบ้าง (จากการโจมตี/เคาน์เตอร์/ผล challenge ที่เปิดเผย)
- `revealed[]` — การ์ดที่ถูกเปิด (จากข่มขู่/แพ้ challenge)
- `seen_via_hack` — การ์ดที่ตัวเองแอบดูด้วยท่านักสืบ
- ตัวนับการ์ดสาธารณะ: อาชีพ X ถูกเปิดทิ้ง/เผยไปแล้วกี่ใบจาก 3 ใบ

**สูตรความมั่นใจว่า "เป้ากำลังโกหก"** (ใช้ตัดสิน challenge):
```
suspicion = base 0.30
+0.30 ถ้าเป้าเคยอ้างอาชีพอื่นไปแล้ว ≥ 2 อาชีพในเกมนี้ (อ้างเกินมือที่ถือได้)
+0.40 ถ้าอาชีพที่เป้าอ้าง ถูกเปิดสาธารณะครบ 3 ใบแล้ว (เป็นไปไม่ได้ — challenge แน่นอน)
+0.25 ถ้าตัวเองถืออาชีพนั้นครบตามจำนวนที่เหลือในระบบ
+0.20 ถ้าเคย hack เห็นการ์ดเป้าแล้วไม่ใช่อาชีพที่อ้าง
-0.20 ถ้าเป้าเพิ่งชนะ challenge มาในเกมนี้ (น่าเชื่อถือขึ้น)
→ challenge เมื่อ suspicion ≥ threshold ของบุคลิก
```

### 7.4 ตรรกะการตัดสินใจต่อจุด

**เฟสโจมตี:**
1. เลือกว่าโจมตีหรือข้าม ตาม aggression ของบุคลิก (ปรับขึ้นถ้าตกอันดับท้าย, ปรับลงถ้าติดเครดิตพัง)
2. เลือกท่า: ให้คะแนนทุกท่าที่ *เข้าเงื่อนไขกับเป้าที่มองเห็น* — ท่าที่ตัวเองถือจริงได้โบนัสคะแนน, ท่าที่ต้องโกหกโดนหักตาม (1 − อัตราโกหก)
3. เลือกเป้า: ถ่วงน้ำหนักด้วย อันดับหยาบ (สูง=น่าตี), ความแค้น (ใครเคยตีฉัน +คะแนน), ข้อมูลที่มี (เคยส่องพอร์ต = เล็งท่าเฉพาะทางได้แม่น)
4. ท่าที่มีเงื่อนไขมองไม่เห็น (เทใส่/บีบหนี้/ฮุบที่ดิน ต้องรู้พอร์ต-หนี้เป้า): บอทใช้ได้เฉพาะกับเป้าที่ *เคยส่องหรือ hack แล้ว* หรือเดาจากพฤติกรรม (เห็น avatar เข้าศูนย์คริปโตบ่อย → เชื่อว่าถือคริปโต) — ห้าม engine เผยข้อมูลให้บอทตรงๆ

**ตอบโต้ใน 10 วิ (ยอม/challenge/counter):**
1. ถ้าถืออาชีพเคาน์เตอร์จริง → counter เสมอ
2. ถ้าไม่ถือ: พิจารณา counter มั่ว เมื่อ (มูลค่าความเสียหายจากท่า > $800×1.5) และสุ่มผ่านอัตราโกหก
3. ไม่งั้นคำนวณ suspicion → challenge ถ้าถึง threshold
4. ไม่เข้าเงื่อนไขไหนเลย → ยอมโดน
- บอทตอบภายใน 2–6 วิ (สุ่ม) ให้ฟีลเหมือนคนคิด ไม่ตอบ 0 วิ

**เฟสแมพ 60 วิ:**
- บอทวางแผนคิว 2–4 กิจกรรมตามบุคลิก แล้วเดินตามคิวด้วยความเร็ว avatar ปกติ (แชร์ระบบเดิน/เวลาเดียวกับผู้เล่น — ห้าม teleport)
- ตรรกะลงทุนพื้นฐาน: มีข่าววงในถือไว้ → ซื้อ/ขายตามข่าวก่อนรอบครบกำหนด (นักสังเกตการณ์จะ fact-check ก่อนถ้าเงินพอ, นักพนันเชื่อทันที)
- ไม่มีข่าว → ปรับพอร์ตเข้ารูปตามบุคลิก (เต่าซื้อทอง, ฉลามซื้อตัวที่เพิ่งขยับแรง)
- โควตาพฤติกรรมหลอก: ทุกบอทมีโอกาส 0.15 ต่อเทิร์นที่จะเดินเข้าตึกแบบไม่ทำธุรกรรม 1 ครั้ง

### 7.5 เรื่อง implement ที่ต่างจาก multiplayer
- เทิร์นบอทบีบเวลาได้: เฟสแมพบอทไม่ต้องรอเต็ม 60 วิ — จบคิวกิจกรรมแล้วข้ามเทิร์นเลย (แต่ animation เดินยังโชว์ให้ผู้เล่นเห็นเส้นทาง เพราะเป็นข้อมูลสำคัญของเกม)
- popup 10 วิของผู้เล่นจริงทำงานปกติแม้ผู้โจมตีเป็นบอท
- โครงโค้ดควรแยก `PlayerController` / `BotController` ที่คุยกับ game engine ผ่าน interface เดียวกัน — วันหลังสลับ BotController เป็น network player ได้ทันที ไม่ต้องรื้อ engine
