# 🌱 ส่วน STARTUP HUB & GREEN INVEST
## เอกสารออกแบบระบบย่อยของเกม (Module Design Document) — v0.2
*อัปเดตล่าสุด: 7 กรกฎาคม 2026 · สถานะ: คอนเซปต์ พร้อมทำ prototype*

> **⚠️ ขอบเขตเอกสาร:** ออกแบบเฉพาะ *ส่วน Startup Hub + Green Invest* ซึ่งเป็นโหมด/พื้นที่หนึ่ง**ภายในเกมหลัก** — ไม่ใช่เกมแยก จุดที่ต้องต่อสายกับเกมหลักติดป้าย 🔗 ไว้ทั่วเอกสาร *(v0.2 ปรับสโคปจากดราฟต์เดิมที่เขียนไว้เป็นเกมเดี่ยว)*

## 1. ภาพรวมของส่วนนี้

**Pitch ประโยคเดียว:** ส่วนของเกมที่ผู้เล่นเข้ามาเปิดบริษัทของตัวเองในโซน **Startup Hub** แล้วใช้โซน **Green Invest** ลงทุนอัปเกรดความกรีนให้ธุรกิจ — โดยความกรีนไม่ใช่แต้มบุญ แต่คือ **กุญแจเปิดตลาดใหม่** และ **เกราะกันดราม่า**

| หัวข้อ | รายละเอียด |
|---|---|
| บทบาทในเกมหลัก | โหมด/พื้นที่จำลองธุรกิจ: ผู้เล่นเข้าโซน Startup Hub เพื่อเปิด-บริหารบริษัทของตัวเอง และสลับไปโซน Green Invest เพื่อลงทุนความกรีน |
| 🔗 จุดเชื่อมกับเกมหลัก | ทางเข้า-ออกโหมด · เงินที่ใช้ (แชร์กับเกมหลักหรือแยกกระเป๋า — หมวด 16 ข้อ 1) · รางวัลที่ส่งกลับเมื่อจบ (หมวด 13) |
| ความยาว 1 รอบของส่วนนี้ | แคมเปญ 36 เทิร์น (1 เทิร์น = 1 เดือนในเกม = 3 ปี) — ย่อ/ขยายให้เข้ากับจังหวะของเกมหลักได้ |
| จุดขายของส่วนนี้ | ระบบใบรับรอง (Consent) ที่ทำหน้าที่เป็น "กุญแจตลาด" + ระบบ Rumors ที่ให้สังคมในเกมเป็นคนบีบให้กรีน ไม่ใช่ระบบบังคับ |

**ปรัชญาการออกแบบ 3 ข้อ**

1. **ผู้เล่นเลือกเองว่าจะกรีนแค่ไหน** — เกมไม่ห้ามสายเทา แต่ออกแบบให้ระยะยาวสายเขียวคุ้มกว่าเสมอ (ผ่านต้นทุน, ตลาด, และดราม่า)
2. **ทุกการลงทุนกรีนต้องมี dual payoff** — ลดคาร์บอน + ให้ผลทางธุรกิจอย่างน้อย 1 อย่าง ผู้เล่นสายคำนวณจะได้อยากซื้อเอง
3. **ยิ่งโต ยิ่งหนีความกรีนไม่พ้น** — บริษัทเล็กไม่มีใครสนใจ แต่พอเข้าห้าง/ส่งออก สายตาสังคมจะไล่ตามทันที

## 2. Core Loop

```
        ┌─────────────────────────────────────────────┐
        │                                             │
        ▼                                             │
  🏢 STARTUP HUB                 🌱 GREEN INVEST      │
  ทำธุรกิจ → ได้เงิน + คาร์บอน ──▶ เอาเงินไปลงทุนกรีน   │
        ▲                        ลดคาร์บอน/สร้างเครดิต │
        │                                │            │
        │      🔑 CONSENT (ใบรับรอง)      │            │
        └── ปลดล็อกตลาดใหม่ ◀── เงื่อนไขครบ ┘            │
                 │                                    │
                 ▼                                    │
        ยอดผลิตพุ่ง → คาร์บอนรวมพุ่ง → เกณฑ์ใบเริ่มหวิดตก ┘
                 (วนกลับไป Green Invest รอบใหม่)
```

**กติกาแบ่งหน้าจอ (จำข้อเดียว):** เงินที่ใช้เพื่อ *หาเงิน* → อยู่หน้าต่าง **Startup Hub** / เงินที่ใช้เพื่อ *ลดคาร์บอนหรือสร้างเครดิตกรีน* → อยู่หน้าต่าง **Green Invest**

## 3. ค่ากลางบน HUD (เห็นตลอดทุกหน้าต่าง)

| ไอคอน | ค่า | นิยาม | ช่วงค่า |
|---|---|---|---|
| 💰 | เงินสด | เงินหมุนเวียนของบริษัท | ≥ 0 (ติดลบ 2 เทิร์นติด = ล้มละลาย) |
| 🌫️ | ดัชนีคาร์บอน/หน่วย | คาร์บอนต่อสินค้า 1 ชิ้น เทียบวันเปิดร้าน (เริ่ม 100%) | 0–100%+ |
| 🏭 | คาร์บอนรวม/เทิร์น | ดัชนี × ปริมาณผลิต (หน่วย "cu") — ใช้กับภาษีคาร์บอนและเป้ารวมของฮับ | ไม่จำกัด |
| 👁️ | สายตาสังคม | สังคมจับตาบริษัทเราแค่ไหน โตตามตลาดสูงสุดที่เปิด | 0–100 |
| ⭐ | ชื่อเสียง | ภาพลักษณ์แบรนด์ ขยับจากเหตุการณ์/แคมเปญ มีผลคูณยอดขาย | −100 ถึง +100 |
| 🗓️ | เทิร์น | เดือนปัจจุบัน / 36 | 1–36 |

**ค่าสายตาสังคมตามตลาด:** ตลาดชุมชน 20 · ตลาดนัดกรีน 30 · ห้าง 55 · B2B 75 · ส่งออก 100 (บวกเพิ่ม +10 ถาวรถ้าเคยเป็นข่าวดัง ไม่ว่าดีหรือร้าย)

> 🔗 **ต่อสายกับเกมหลัก:** 💰 เงินสดจะใช้สกุลเดียวกับเกมหลักเลย หรือแลกเข้า-ออกเป็น "ทุนธุรกิจ" ตอนเข้าโหมด — ต้องเคาะ (คำถามค้างข้อ 1 หมวด 16) ส่วนค่าที่เหลือ (คาร์บอน สายตาสังคม ชื่อเสียง เทิร์น) เป็นค่าภายในของส่วนนี้ ไม่รั่วออกไปเกมหลัก

> ⚠️ จุดเทคนิคสำคัญ: เกณฑ์ใบรับรองทุกใบวัดจาก **ดัชนีคาร์บอนต่อหน่วย** ไม่ใช่คาร์บอนรวม — เกมจะได้ไม่ลงโทษคนขายดี แต่บีบให้ "ผลิตสะอาดขึ้น" แทน ส่วนคาร์บอนรวมเอาไว้ใช้กับภาษีคาร์บอนและสายตาสังคม

## 5. หน้าต่างที่ 1: 🏢 STARTUP HUB (โซนทำธุรกิจ)

### 5.1 ร้านของฉัน (My Business) — จอหลัก
โชว์เลเวลร้าน กำลังผลิต ต้นทุน/ชิ้น ราคาขาย และ **อัปเกรดสายธุรกิจล้วน** ตัวอย่างของคาเฟ่:

| อัปเกรด | ราคา | ผล |
|---|---|---|
| เครื่องชงตัวที่ 2 | ฿30K | เพดานผลิต +800 แก้ว |
| จ้างบาริสต้า | ฿6K/เทิร์น | เพดานผลิต +400 แก้ว + ปลดล็อกเมนูพิเศษ |
| ตกแต่งร้านใหม่ | ฿20K | ชื่อเสียง +10, ฐานลูกค้า +10% |
| ออกเมนูซิกเนเจอร์ | ฿10K | กำไร/แก้ว +฿5 |
| สาขาที่ 2 (ปลายเกม) | ฿150K | ฐานลูกค้า ×1.8 แต่ fixed +฿15K และคาร์บอนรวม ×1.8 |

*(ธุรกิจอื่นใช้แพตเทิร์นเดียวกัน: เพิ่มเพดานผลิต / เพิ่มฐานลูกค้า / เพิ่ม margin / เพิ่มชื่อเสียง)*

### 5.2 ตลาด (Market)
ตลาดวางเป็นชั้น แต่ละชั้น **ล็อกด้วยใบรับรอง** จากหน้าต่าง Green Invest (ดูรายละเอียดหมวด 8)

### 5.3 เพื่อนบ้านในฮับ (Community)
โต๊ะดีลกับบริษัทข้างห้อง — ขาย by-product, แชร์รอบขนส่ง, ออกบูธงานฮับ (ดูหมวด 11)

### 5.4 รายงาน & ข่าว (Dashboard)
สรุปรายรับ-จ่ายต่อเทิร์น กราฟยอดขาย และ **news feed** ที่ระบบ Rumors จะเด้งข่าวดี/ร้ายใส่หน้าเรา

## 6. หน้าต่างที่ 2: 🌱 GREEN INVEST (โซนลดคาร์บอน + ปั๊ม Consent)

### 6.1 มิเตอร์คาร์บอน (Carbon Dashboard) — จอหลัก
แตก footprint เป็น 4 หมวด (พลังงาน/วัสดุ/ของเสีย/ขนส่ง) เห็นชัดว่ารั่วหมวดไหน ควรเทเงินตรงไหนก่อน

### 6.2 แคตตาล็อกลงทุนกรีน
ชั้นวางของแบ่งตาม 4 หมวด ทุกใบโชว์ 3 ตัวเลข: **ราคา / คาร์บอนที่ลด / ผลพลอยได้ทางธุรกิจ** (แคตตาล็อกเต็มอยู่หมวด 7)

### 6.3 ศูนย์ใบรับรอง (Consent Center)
บันไดใบรับรอง + checklist เรียลไทม์ว่าใบถัดไปขาดอะไรอีกกี่ข้อ ครบแล้วกดสมัคร → จ่ายค่าธรรมเนียม → รอ audit

### 6.4 แหล่งเงินสีเขียว (Green Finance)
ปลดล็อกตามระดับใบ: green loan ดอกเบี้ยถูก → เงินอุดหนุนรัฐ → impact investor ขอถือหุ้น

### 6.5 ป้ายประกาศความกรีน (Green Claims)
ยิงแคมเปญ "เราลดคาร์บอนแล้ว X%" — วางเมนูนี้ไว้ฝั่ง Green Invest **จงใจ** เพื่อให้เกมเทียบคำโฆษณากับมิเตอร์จริงได้ทันที (กติกาอยู่หมวด 10.4)

## 7. 📗 แคตตาล็อกลงทุนกรีนฉบับเต็ม

**กติกาออกแบบ 4 ข้อ**
1. ทุกชิ้นมี dual payoff (คาร์บอน + ธุรกิจ)
2. แบ่ง 3 Tier: T1 ถูก-คืนทุนไว (สอนว่ากรีน=คุ้ม) / T2 ต้องชั่งใจกับการขยายธุรกิจ / T3 แพง-ผูกใบรับรองสูง
3. **เช่าของฮับ vs ซื้อเอง** — เช่าถูกกว่าตอนต้นเกม แต่ใบ Lv.3+ นับเฉพาะของที่เป็นเจ้าของ → เกิดจังหวะ "เช่าก่อน รีบเปิดตลาด แล้วค่อยซื้อจริง"
4. เปอร์เซ็นต์ลด = ลด *ภายในหมวด* และของหมวดเดียวกัน **ซ้อนแบบคูณ** (−30% สองชิ้น = 0.7×0.7 ≈ −51%)

## 8. 🔑 ระบบใบรับรอง (Consent) → ขยาย Market

สูตรเงื่อนไขเดียวกันทุกใบ: **อัปเกรดของจริงครบเกณฑ์ + ดัชนีคาร์บอน/หน่วยต่ำกว่าเส้น + ค่าธรรมเนียม + ผ่าน audit**

| ใบ | เงื่อนไข | ค่าสมัคร | เวลา audit | ปลดล็อก |
|---|---|---|---|---|
| **Lv.1 ใบรับรองกรีนของฮับ** | อัปเกรดกรีน ≥ 2 ชิ้น + ดัชนี ≤ 85% | ฿5K | 1 เทิร์น | ตลาดนัดกรีน + อีเวนต์ฮับ → ฐานลูกค้า +25% |
| **Lv.2 ฉลากเขียว (รัฐ)** | ครอบคลุม ≥ 3 หมวด + ดัชนี ≤ 65% + ไม่มีธง greenwash ค้าง | ฿15K | 1 เทิร์น | ห้าง/ซูเปอร์ → ฐาน ×2 + ลดหย่อนภาษี ฿3K/เทิร์น |
| **Lv.3 ฉลากคาร์บอน** | ดัชนี ≤ 45% + มีพลังงานสะอาด (โซลาร์เอง หรือ RE100) + Sustainability Officer + Carbon tracking | ฿30K | 2 เทิร์น | ลูกค้า B2B → ออเดอร์คงที่ +฿20K/เทิร์น + สิทธิ์ green loan |
| **Lv.4 มาตรฐานสากล** | ครบ 4 หมวด + ดัชนี ≤ 25% + supply chain audit + RE100 | ฿60K | ผ่านตรวจ 2 เทิร์นติด | ตลาดส่งออก → ราคาขาย +30% + impact investor เข้าหา |

> 📝 หมายเหตุ rebalance: เส้นเกณฑ์ปรับจากดราฟต์เดิม (80/60/40/25) เป็น **85/65/45/25** หลังลองกางตัวเลขจริงในหมวด 14 แล้วพบว่าใบแรกโหดเกินสำหรับ 6 เทิร์นแรก

**กติกาที่ทำให้ใบรับรอง "มีชีวิต" ไม่ใช่ซื้อครั้งเดียวจบ**
- 🎲 **Spot audit สุ่มตรวจซ้ำ** ทุก 4–6 เทิร์นต่อใบ — ถ้าดัชนีเกินเกณฑ์: **ใบโดนพัก + ตลาดที่ผูกอยู่แช่แข็งทันที** ยอดขายส่วนนั้นหายจนกว่าจะแก้แล้วจ่ายตรวจซ้ำ ฿8K
- ใบระดับสูงครอบสิทธิ์ใบล่างทั้งหมด (ถือใบเดียวพอ)
- ลูปหลักของเกม: ปลดตลาดใหม่ → ผลิตพุ่ง → คาร์บอนรวมพุ่ง + สายตาสังคมขึ้น → เกณฑ์เริ่มหวิดตก → กลับไป Green Invest รอบใหม่

## 10. 📰 ระบบเหตุการณ์ & Rumors

### 10.1 สูตรโอกาสดราม่า (ทอยทุกเทิร์นในเฟส 3)

```
โอกาสดราม่า = 15% × (ดัชนีคาร์บอน ÷ 100) × (สายตาสังคม ÷ 100)
              × 3 ถ้าติดธง greenwashing   [เพดาน 45%]
```

ตัวอย่าง: ร้านเข้าห้างแล้ว (สายตา 55) ดัชนี 90% → 15 × 0.9 × 0.55 ≈ **7.4%/เทิร์น** แต่ถ้าเพิ่งโม้แคมเปญเกินจริง → **22%/เทิร์น**

### 10.2 ตารางเหตุการณ์ลบ (สุ่มเมื่อดราม่า trigger)

| เหตุการณ์ | ผล | วิธีกัน/แก้ |
|---|---|---|
| 📢 ม็อบประท้วงหน้าร้าน | ยอด −30% นาน 2 เทิร์น | มี Sustainability Officer → เหลือ −15% |
| #️⃣ ติดแฮชแท็กแบนในโซเชียล | ชื่อเสียง −25, ยอด −15% นาน 3 เทิร์น | ยิงแคมเปญขอโทษ+แผนแก้ (฿10K) หยุดผลทันที |
| 📊 สื่อจัดอันดับ "แบรนด์คาร์บอนสูงสุด" | ตลาด B2B ปิด 2 เทิร์น | ดัชนี ≤ 65% = ไม่มีสิทธิ์ติดอันดับ |
| 🏬 ห้างขีดเส้นตาย | ต้องได้ใบ Lv.2 ภายใน 4 เทิร์น ไม่งั้นถูกถอดจากชั้น | (เหตุการณ์นี้เกิดเฉพาะคนเข้าห้างทาง "โปรโมชันพิเศษ" โดยไม่มีใบ) |
| 😔 พนักงานรุ่นใหม่ทยอยลาออก | ค่าใช้จ่ายคงที่ +20% จนกว่าชื่อเสียง ≥ 0 | ตกแต่งร้าน/แคมเปญกู้ชื่อเสียง |
| 🗑️ ลูกค้ารีวิวขยะล้นหลังร้าน | ยอด −10% นาน 2 เทิร์น | มีถังแยก+ถังหมัก = ภูมิคุ้มกันเหตุการณ์นี้ |
| 🚨 โดนแฉ greenwashing | ยอด −40% นาน 3 เทิร์น, ใบรับรองพัก, ชื่อเสียง −40 | เกิดเฉพาะคนติดธง (ดู 10.4) |
| ⚡ ค่าไฟพีค/ไฟดับ | fixed +฿5K เทิร์นนั้น | มีแบตเตอรี่กักเก็บ = immune |

**เหตุการณ์กฎหมาย (ไม่สุ่ม — มาแน่กลางเกม):** 🏛️ *ภาษีคาร์บอนประกาศใช้เทิร์นที่ 18* → ทุกบริษัทจ่าย ฿0.8K ต่อคาร์บอนรวม 100 cu ที่เกินโควตาฟรี 300 cu/เทิร์น — ประกาศเตือนล่วงหน้าตั้งแต่เทิร์น 12 ให้ผู้เล่นเตรียมตัว (เลียนแบบ CBAM/พ.ร.บ. Climate Change ของจริง)

### 10.3 ตารางเหตุการณ์บวก (สุ่มเมื่อดัชนี ≤ 60% หรือชื่อเสียง ≥ 20)

| เหตุการณ์ | ผล |
|---|---|
| 🤳 Influencer สายกรีนรีวิวให้ฟรี | ยอด +25% นาน 2 เทิร์น |
| 💼 Impact fund เสนอเงินลงทุน | รับ ฿50K แลกหุ้น 5% (เลือกรับ/ไม่รับ — กระทบคะแนนมูลค่าบริษัทตอนจบ) |

| 🏛️ รัฐอุดหนุนอุปกรณ์กรีน | ของในแคตตาล็อกลด 50% เทิร์นนี้ |
| 🎤 ฮับเชิญขึ้นเวที showcase | ชื่อเสียง +15 + เปิดดีลเพื่อนบ้านใหม่ 1 ดีล |
| 🧳 นักท่องเที่ยวสายยั่งยืนมาลง content | ฐานลูกค้าถาวร +5% |

### 10.4 กติกา Green Claims (ดาบสองคม)

- ยิงแคมเปญ "เราลดคาร์บอน X%" ราคา ฿12K → ยอด +20% นาน 3 เทิร์น
- ถ้ามี Carbon tracking software **และ** ตัวเลขจริง ≥ X → ปลอดภัย 100%
- ถ้าโม้เกินมิเตอร์จริง → ติด **ธง greenwashing** เงียบๆ (ผู้เล่นเห็น ไอคอนธงเหลืองบน HUD): โอกาสดราม่า ×3 จนกว่าตัวเลขจริงจะตามทัน หรือจนโดนแฉ
- นี่คือการกลับด้านจากเกมภาคนักลงทุน — คราวนี้ **ผู้เล่นเองคือฝ่ายถูกล่อใจให้ฟอกเขียว**

### 11.3 เป้าหมายรวมประจำไตรมาส
"ทั้งฮับลดคาร์บอนรวม 500 cu ภายใน 3 เทิร์น" → สำเร็จ = สมาชิกทุกคนได้โบนัส (เงิน/ชื่อเสียง/ส่วนลด) — สร้างความรู้สึกทีมเดียวกัน และเป็น hook ธรรมชาติสำหรับโหมด multiplayer ในอนาคต

### 11.4 อีเวนต์ของฮับ
ออกบูธงานฮับ ฿3K/ครั้ง → ฐานลูกค้าถาวร +2% (ทำซ้ำได้ทุก 4 เทิร์น)

## 12. 🧮 สูตรคำนวณหลัก (สำหรับคนทำ prototype)

```
ยอดขาย/เทิร์น = min( ฐานลูกค้า × ตัวคูณตลาด × (1 + ชื่อเสียง/500) × ตัวคูณเหตุการณ์ ,
                     เพดานผลิต )

กำไรสุทธิ    = ยอดขาย × กำไรต่อหน่วย
              + รายได้เสริม (ขายปุ๋ย, ดีล by-product, ขายไฟ, ลดหย่อนภาษี)
              − ค่าใช้จ่ายคงที่ − สัญญารายเทิร์น − ภาษีคาร์บอน (หลังเทิร์น 18)

ดัชนี CO₂/หน่วย = Σ ทุกหมวด [ คะแนนฐานหมวด × Π (1 − %ลดของไอเท็มในหมวด) ]

คาร์บอนรวม (cu) = ดัชนี × ยอดผลิต ÷ 100

ภาษีคาร์บอน     = ฿0.8K × max(0, คาร์บอนรวม − 300) ÷ 100

โอกาสดราม่า    = min(45%, 15% × ดัชนี/100 × สายตาสังคม/100 × [3 ถ้าติดธง greenwash])
```

## 13. 🏁 การจบของส่วนนี้ & สิ่งที่ส่งกลับเกมหลัก

**โหมดแคมเปญ:** จบที่เทิร์น 36 (ครบ 3 ปี) — **โหมดแซนด์บ็อกซ์:** เล่นต่อไม่จำกัด (จะใช้แบบไหนขึ้นกับดีไซน์เกมหลัก)

**Fail state:** เงินสดติดลบ 2 เทิร์นติดกัน = ล้มละลาย (ก่อนล้มมีตัวช่วย: green loan ถ้ามีใบ Lv.3, หรือขายหุ้นให้ impact fund) — ล้มแล้วออกจากโหมดกลับสู่เกมหลัก และกลับมาเปิดบริษัทใหม่ได้

**คะแนนรวม 4 แกน:**

```
คะแนน = (มูลค่าบริษัท ÷ ฿10K)          ← เงินสด + ทรัพย์สินกรีนที่ซื้อขาด
      + (100 − ดัชนีคาร์บอน) × 2       ← ยิ่งสะอาดยิ่งได้
      + ระดับใบสูงสุด × 50
      + ชื่อเสียง
      − จำนวนครั้งที่โดนแฉ greenwashing × 30
```

| คะแนน | ฉายา |
|---|---|
| < 150 | 🐣 เถ้าแก่ฝึกหัด |
| 150–299 | 🌱 SME สายพันธุ์ใหม่ |
| 300–449 | 💚 นักธุรกิจหัวใจสีเขียว |
| 450–599 | 🦄 กรีนยูนิคอร์น |
| ≥ 600 | 🌍 ตำนานแห่ง GreenHub |

**เส้นชัยลับ "Green IPO":** ได้ใบ Lv.4 + มูลค่าบริษัท ≥ ฿1M + ชื่อเสียง ≥ 50 ก่อนเทิร์น 36 → ฉากจบพิเศษ บริษัทเข้าตลาดหุ้นสายความยั่งยืน

> 🔗 **สิ่งที่ส่งกลับเกมหลักเมื่อจบ (เลือก/ผสมได้ตามดีไซน์เกมหลัก):**
> - แปลงคะแนนรวมเป็นเงิน ไอเทม หรือแต้มพัฒนาในเกมหลัก
> - ฉายาที่ได้กลายเป็น achievement / title ติดตัวผู้เล่น
> - บริษัทที่ปั้นสำเร็จกลายเป็น **asset ถาวรในโลกเกมหลัก** — สร้าง passive income เข้ากระเป๋าผู้เล่นเรื่อยๆ และตัวอาคารในโซน Startup Hub เปลี่ยนหน้าตาตามระดับใบรับรองที่ทำได้ (โชว์ให้ผู้เล่นคนอื่นเห็น)
> - แต้มกรีนสะสมของทุกบริษัทในโซน อาจไปขับ "ความเขียวของเมือง/โลกในเกมหลัก" ต่อ — เชื่อมโมดูลนี้เข้ากับภาพใหญ่ของเกมได้พอดี

## 16. ❓ คำถามที่ยังเปิดอยู่ (ตัดสินใจก่อนเริ่ม MVP)

1. 🔗 **เงินแชร์กับเกมหลักหรือแยกกระเป๋า** — ใช้สกุลเดียวกันเลย (เชื่อมแน่น แต่ balance ยาก เพราะเงินจากส่วนอื่นของเกมจะทะลักเข้ามา) หรือแลกเป็น "ทุนธุรกิจ" ตอนเข้าโหมดด้วยเรตคงที่ (คุม balance ของโมดูลได้เอง — *แนะนำเริ่มแบบนี้ก่อน*)
2. 🔗 **จังหวะการเล่น** — ผู้เล่นนั่งเล่นรวดเดียวจนจบแคมเปญ หรือแวะเข้าโซนมาเดินหน้าทีละไม่กี่เทิร์นแล้วออกไปทำกิจกรรมอื่นในเกมหลัก (แบบหลังต้องมี save state ของบริษัทต่อผู้เล่น และเข้ากับเกมหลักแนวเปิดโลก/หลายโซนมากกว่า)
3. **Green Invest เป็น "แท็บเมนู" หรือ "สถานที่จริงในโซน"** (เคาน์เตอร์ธนาคารเขียว, สำนักงานที่ปรึกษา, NPC auditor เดินมาตรวจถึงร้าน) — แบบแรกคลีน/ทำไว แบบหลังกลืนกับเกมหลักที่เป็นโลกให้เดินสำรวจมากกว่า → *กระทบสโคป MVP มากที่สุด*
4. โทนภาพ — ยึดตามอาร์ตของเกมหลัก (โมดูลนี้ไม่ควรกำหนดเอง)
5. ชื่อเรียกส่วนนี้ในเกม — working title: *โซน GreenHub (Startup Hub × Green Invest)*
6. ระบบ AP เอาไว้หรือตัดออก (รอผล playtest MVP)
